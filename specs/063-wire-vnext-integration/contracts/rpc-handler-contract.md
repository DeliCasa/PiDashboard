# RPC Handler Contract: Wire Factory Integration

**Feature**: 063-wire-vnext-integration
**Date**: 2026-03-06

## Overview

This contract defines the interface between wire package factory functions and MSW RPC handlers. No new API endpoints are introduced — this documents the test-infrastructure contract.

## Handler Factory Signatures (Updated)

### `createRpcSessionHandlers(options?)`

**Endpoints intercepted**:
- `POST /rpc/delicasa.device.v1.SessionService/ListSessions`
- `POST /rpc/delicasa.device.v1.SessionService/GetSession`

**Input**: Optional overrides for session factory defaults. Accepts an array of session overrides or pre-built session objects.

**Output**: MSW `HttpHandler[]` returning proto-JSON generated by `makeListSessionsResponse()` and `makeGetSessionResponse()`.

### `createRpcEvidenceHandlers(options?)`

**Endpoints intercepted**:
- `POST /rpc/delicasa.device.v1.EvidenceService/GetSessionEvidence`
- `POST /rpc/delicasa.device.v1.EvidenceService/GetEvidencePair`

**Input**: Optional overrides for evidence factory defaults.

**Output**: MSW `HttpHandler[]` returning proto-JSON generated by `makeGetSessionEvidenceResponse()` and `makeGetEvidencePairResponse()`.

### `createRpcCameraHandlers(options?)`

**Endpoints intercepted**:
- `POST /rpc/delicasa.device.v1.CameraService/ListCameras`
- `POST /rpc/delicasa.device.v1.CameraService/GetCamera`

**Input**: Optional overrides for camera factory defaults.

**Output**: MSW `HttpHandler[]` returning proto-JSON generated by `makeListCamerasResponse()` and `makeGetCameraResponse()`.

### `rpcErrorHandlers`

**Preset error scenarios** (unchanged interface, same keys):
- `sessionsUnavailable` — returns Connect error code 14 (UNAVAILABLE)
- `sessionsEmpty` — returns empty list response
- `sessionNotFound` — returns Connect error code 5 (NOT_FOUND)
- `evidenceEmpty` — returns empty captures response

## Transport Unit Test Contract

### `transport.ts` fetch wrapper

**Contract**: The custom fetch wrapper in `createConnectTransport()` MUST:
1. Strip `signal` from `RequestInit` when present
2. Pass `RequestInit` through unmodified when `signal` is absent
3. Late-bind to `globalThis.fetch` (not capture at module load time)

**Test location**: `tests/unit/rpc/transport.test.ts`
